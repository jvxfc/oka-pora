# 🌿 Oka Pora

Projeto desenvolvido durante o primeiro módulo do IFSP com foco no aprendizado de HTML e CSS.

## 📚 Sobre o Projeto

O Oka Pora foi criado como atividade prática para desenvolver conhecimentos básicos de Front-End, utilizando apenas tecnologias fundamentais da web.

Durante o desenvolvimento foram aplicados conceitos como:

- Estruturação de páginas com HTML5;
- Estilização com CSS3;
- Organização de layouts;
- Uso de cores, fontes e espaçamentos;
- Responsividade básica;
- Boas práticas iniciais de desenvolvimento web.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3

## 💻 Objetivo

O principal objetivo do projeto foi colocar em prática os primeiros conceitos de desenvolvimento web aprendidos no curso, servindo como base para projetos futuros mais avançados.
